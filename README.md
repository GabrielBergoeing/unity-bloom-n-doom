# unity-bloom-n-doom
Official repository of the unity version of Bloom &amp; Doom

## Requirements
**Unity Version:** 6000.0.40f1

- [x] Player can water the plants
- [x] Player can let watered plants grow
- [ ] Player can sabotage with flamethrowers

- [x] Player can shoot water gun to push other players
- [x] Player can depend on the water resource by losing it
- [x] Player can manage his inventory with seeds, tools and fertilizer

- [x] Player can play a basic match with another player
